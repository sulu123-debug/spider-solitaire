# 找不同 🔍

找出两张图片中的不同之处，考验你的眼力！

## 🎮 游戏规则

1. **目标**：在限定时间内找出两张图片中所有的不同之处

2. **玩法**：
   - 左右两边会显示略微不同的图片
   - 点击图片中有差异的地方
   - 找出所有差异点即可过关

3. **难度递进**：
   - 关卡 1-3：3处不同
   - 关卡 4-6：4处不同
   - 关卡 7+：5处不同

## 🎯 版本说明

本游戏提供两个版本：

### V2 精美版（推荐）
- **多主题系统**：电影、水果、自然、动物、美食、运动
- **关卡系统**：每主题10关，难度递增
- **精美UI**：使用 Vant UI 组件库
- **庆祝动画**：通关时 Confetti 动画效果
- **功能按钮**：提示、加时、放大

### 原版
- 经典玩法
- 简洁界面
- 适合怀旧玩家

## 🖱️ 操作方式

| 操作 | 方法 |
|------|------|
| 找出不同 | 点击左右图片中的差异点 |
| 提示 | 点击提示按钮显示一个答案位置 |
| 加时 | 点击加时按钮增加时间 |
| 放大 | 点击放大按钮查看细节 |

## 📊 计分规则

- 找出所有差异点获得基础分
- 用时越短奖励越高
- 剩余时间转化为额外分数
