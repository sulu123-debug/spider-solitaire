# 蜘蛛纸牌 🕷️

经典 Windows 纸牌游戏，将相同花色的牌按从 K 到 A 的顺序排列即可消除。

## 🎮 游戏规则

1. **目标**：将所有牌按同花色从 K 排列到 A，即可消除一组。完成全部 8 组即为胜利。

2. **基本规则**：
   - 可以移动连续同花色的降序牌组（如：红心8→红心7→红心6）
   - 任何牌都可以放在比它大 1 的牌上面（如：黑桃Q可以放在红心K上）
   - 空列可以放置任意牌或牌组
   - 发牌时每列必须至少有一张牌

3. **难度设置**：
   - 🟢 1 花色 - 简单：仅使用黑桃
   - 🟡 2 花色 - 中等：使用黑桃和红心
   - 🔴 4 花色 - 困难：使用全部四种花色

## 🖱️ 操作方式

| 操作 | 方法 |
|------|------|
| 选择纸牌 | 点击纸牌选中，再次点击目标列放置 |
| 移动纸牌 | 拖拽纸牌到目标列 |
| 自动移动 | 双击纸牌自动移动到最佳位置 |
| 整组移动 | 双击同花色连续牌组，整组自动移动 |
| 撤消 | Ctrl + Z 或点击菜单 |
| 提示 | 按 H 键或点击菜单获取提示 |
| 暂停 | 按 P 键暂停/继续游戏 |

## ⌨️ 快捷键

- `Ctrl + Z` - 撤消上一步
- `H` - 获取提示
- `P` - 暂停/继续
- `Esc` - 取消选择/关闭对话框

## 📊 计分规则

- 初始分数：500 分
- 移动一张牌：-1 分
- 消除一组（K到A）：+100 分

## 🏆 排行榜

游戏会自动记录你的最佳成绩，按用时排序显示前10名。

## 🔧 技术特性

- 支持鼠标拖拽和点击操作
- 流畅的动画效果
- 自动保存游戏进度
- 排行榜本地存储
