# 数独挑战 🔢

经典数字益智游戏，在9x9网格中填入1-9数字，使每行、每列、每个3x3区块都不重复。

## 🎮 游戏规则

1. **目标**：在9x9的格子中填入数字1-9，使得：
   - 每一行都包含1-9所有数字
   - 每一列都包含1-9所有数字
   - 每个3x3的九宫格都包含1-9所有数字

2. **难度级别**：
   - 🟢 简单：40个初始数字
   - 🟡 中等：30个初始数字
   - 🔴 困难：25个初始数字
   - 🔴 专家：20个初始数字

## 🖱️ 操作方式

| 操作 | 方法 |
|------|------|
| 选择格子 | 点击任意空格选中 |
| 填写数字 | 点击底部数字按钮填入 |
| 清除数字 | 点击清除按钮或填入0 |
| 高亮同行列 | 点击格子后自动高亮相同数字 |

## 🎯 游戏特性

- **自动检查**：填写后自动检测冲突并显示红色
- **笔记模式**：长按数字按钮进入笔记模式
- **唯一候选数**：自动提示唯一可填数字
- **撤销重做**：支持撤销和重做操作
- **提示功能**：提供当前选中格子的正确数字

## ⌨️ 快捷键

- `1-9` - 填入数字
- `0` 或 `Delete` - 清除格子
- `Ctrl + Z` - 撤销
- `Ctrl + Y` - 重做
- `N` - 切换笔记模式

## 📊 计分规则

- 完成谜题获得基础分
- 用时越短分数越高
- 尝试次数越少分数越高

- 自动保存游戏进度
