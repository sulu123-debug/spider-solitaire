# 扫雷挑战 🚩

经典 Windows 扫雷游戏，左键揭开格子，右键插旗标记地雷。

## 🎮 游戏规则

1. **目标**：找出所有安全格子，避免踩到地雷

2. **数字含义**：格子中的数字表示周围8个格子中有多少个地雷
   - 数字1：周围有1个地雷
   - 数字2：周围有2个地雷
   - 以此类推...

3. **操作**：
   - 左键点击：揭开格子
   - 右键点击：插旗/取消插旗标记地雷
   - 双击数字：快速揭开周围格子（当插旗数等于数字时）

4. **难度级别**：
   - 🟢 简单：9×9格子，10个地雷
   - 🟡 中等：16×16格子，40个地雷
   - 🔴 困难：30×16格子，99个地雷

## 🖱️ 操作方式

| 操作 | 方法 |
|------|------|
| 揭开格子 | 左键点击 |
| 标记地雷 | 右键点击 |
| 快速揭开 | 双击已揭开的数字 |
| 重新开始 | 点击表情按钮 |

## 🎯 游戏特性

- **三种难度**：简单、中等、困难
- **首次点击安全**：保证第一次点击不会踩雷
- **插旗标记**：用旗帜标记确定的地雷
- **问号标记**：用问号标记不确定的格子
- **计时功能**：记录完成用时

## ⌨️ 快捷键

- `F2` - 重新开始
- `Ctrl + Click` - 快速揭开周围格子

## 💡 技巧

1. 先点击角落或边缘，这些位置踩雷概率较低
2. 利用数字推理地雷位置
3. 已确定的格子可以标记旗帜防止误触
4. 遇到1的数字，且周围已插旗等于1，可快速揭开其他格子

## 🔧 技术特性

- 经典的 Windows 扫雷玩法
- 表情按钮显示游戏状态
- 自动记录最佳成绩
- 响应式设计，支持移动端
